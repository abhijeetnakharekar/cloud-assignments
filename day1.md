# What is cloud?
Cloud is a compute resources as rent. Meaning of rent is pay as you go means you have to pay
for the time you have used the resources.

## Advantages of cloud?
1. We can create virtual machine on cloud.
2. We can use this VM whenever we need it and can switch off it when not needed.
3. VM will give us flexibiity of selecting OS, RAM, Hard disk, space, maintainence etc
4. Whenever our VM is running we get free electricity, internet speed from vendors.
5. According to our requirement we can increase or decrease size of ram & hard disk 
	of virtual machine.
6. We don't have to pay for VM whenever it is switched off.
